## 用户接口

------



#### 分页查询

> 通过分页获取用户信息



##### 请求地址

> /user/page



##### 支持格式

>  FormData



##### 请求方式

> GET



##### 请求参数

|   参数   | 必选 | 类型 |         说明         |
| :------: | :--: | :--: | :------------------: |
| pageNum  |  是  | int  |       当前页数       |
| pageSize |  是  | int  | 当前页有多少用户数据 |



##### 返回参数

| 返回字段 | 返回类型 |                             说明                             |
| :------: | :------: | :----------------------------------------------------------: |
|   code   |   int    | 状态码。 `200`：正常；`201`：没有数据或已经是尾页数据；`500`：异常； |
|  messge  |  String  |                          结果信息。                          |
|   data   |  Object  |             返回分页结果。`pageData`：分页数据。             |



------

#### 添加用户

> 添加一名用户，**管理员专用接口**。



##### 请求地址

> /user/add



##### 支持格式

>  JSON



##### 请求方式

> PUT



##### 请求参数

|   参数   | 必选 |  类型  |                     说明                     |
| :------: | :--: | :----: | :------------------------------------------: |
| username |  是  | String |                    用户名                    |
| password |  是  | String |                     密码                     |
|   role   |  否  | String | 角色。`user`|`admin`。不填参数默认为`user`。 |



##### 返回参数

| 返回字段 | 返回类型 |                          说明                          |
| :------: | :------: | :----------------------------------------------------: |
|   code   |   int    | 状态码。 `200`：正常；`406`：已存在用户；`500`：异常； |
|  messge  |  String  |                       结果信息。                       |
|   data   |  Object  |                          null                          |



------

> 获取一条音乐数据





##### 请求地址

##### 支持格式

>  FormData



##### 请求方式

> GET



##### 请求参数

|   参数   | 必选 | 类型 |         说明         |

#### 添加音乐

> 添加一首音乐



##### 请求地址

> /music/add



##### 支持格式

>  FormData



##### 请求方式

> PUT



##### 请求参数

|   参数    | 必选 |  类型  |               说明                |
| :-------: | :--: | :----: | :-------------------------------: |
| musicName |  是  | String |              音乐名               |
|  author   |  是  | String |               作者                |
| coverFile |  是  | String | 封面文件。目前仅支持`jpg,png`格式 |
| musicFile |  是  | String |   音乐文件。目前仅支持`mp3`格式   |



##### 返回参数

| 返回字段 | 返回类型 |                          说明                          |
| :------: | :------: | :----------------------------------------------------: |
|   code   |   int    | 状态码。 `200`：正常；`406`：已存在用户；`500`：异常； |
|  messge  |  String  |                       结果信息。                       |
|   data   |  Object  |         结果。 `musicAddMsg`：所添加音乐的信息         |
